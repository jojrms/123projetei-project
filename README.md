# PORTUGUÊS

# SOBRE O PROJETO
Este código simula uma área de login, com o acesso pré-definido, onde, caso dê sucesso, direciona o usuário a um dashboard, em que poderá visualizar usuários fakes já existentes, criar, editar e excluir.
